# mb_cvd_causal_decomp
Causal decomposition code for "Contribution of Adolescent Movement Behaviours to Socioeconomic Inequalities in Adult Cardiovascular Disease Risk: A Causal Decomposition in a Population-Based Cohort" by Nicholas Grubic, Katerina Maximova, Arnaud Chiolero, Arjumand Siddiqi, Sarah Carsley, Brice Batomen, and Cristian Carmeli.

The code for the g-formula-based causal decomposition is provided in *Causal_decomp_Rcode_20260427.R*.
